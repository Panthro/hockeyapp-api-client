This is the API

appsGet

Gets `Apps`.

URL

https://rink.hockeyapp.net/api/2/apps

HTTP Method

GET

Response Type

Apps

Parameters

appsUploadPost

Upload a new app

URL

https://rink.hockeyapp.net/api/2/apps/upload

HTTP Method

POST

Response Type

App

Parameters

  • Form: Ipa file

    file data of the .ipa for iOS, .app.zip for OS X, or .apk file for Android

  • Form: dsym file

    file data of the .dSYM.zip file (iOS and OS X) or mapping.txt (Android); note that the extension has to be .dsym.zip (case-insensitive) for iOS and OS X and the file name has to be mapping.txt for Android.

  • Form: notes String

    release notes as Textile or Markdown (after 5k characters note are truncated)

  • Form: notes_type Integer

    type of release notes. `0` for Textile, `1` for Markdown

  • Form: notify Integer

    notify testers (can only be set with full-access tokens). `0` to not notify, `1` to notify all testers that can install this app, `2` to notify all testers

  • Form: status Integer

    download status (can only be set with full-access tokens). `1` to not allow users to download the version, `2` to make the version available for download

  • Form: strategy String

    replace or add build with same build number. `add` to add the build as a new build to even if it has the same build number __default__, `replace` to replace to a build with the same build number

  • Form: tags List(String)

    restrict download to comma-separated list of tags

  • Form: teams List(Integer)

    restrict download to comma-separated list of team IDs

  • Form: users List(Integer)

    restrict download to comma-separated list of user IDs

  • Form: mandatory Integer

    set version as mandatory `0` for not mandatory __default__, `1` for mandatory

  • Form: release_type Integer

    set the release type of the app. `0` for beta __default__ `1` for store `2` for alpha `3` for enterprise

  • Form: private Boolean

    set to `true` to enable the private download page (default is `true`)

  • Form: owner_id Integer

    set to the ID of your organization

  • Form: commit_sha String

    set to the git commit sha for this build

  • Form: build_server_url String

    set to the URL of the build job on your build server

  • Form: repository_url String

    set to your source repository
