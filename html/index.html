post /apps/upload
(appsUploadPost)
Upload a new app

Consumes

This API call consumes the following media types via the Content-Type request header:
  • multipart/form-data

Form parameters

Ipa (required)
Form Parameter — file data of the .ipa for iOS, .app.zip for OS X, or .apk file for Android
dsym (optional)
Form Parameter — file data of the .dSYM.zip file (iOS and OS X) or mapping.txt (Android); note that the extension has to be .dsym.zip (case-insensitive) for iOS and OS X and the file name has to be mapping.txt for Android.
notes (optional)
Form Parameter — release notes as Textile or Markdown (after 5k characters note are truncated)
notes_type (optional)
Form Parameter — type of release notes. `0` for Textile, `1` for Markdown
notify (optional)
Form Parameter — notify testers (can only be set with full-access tokens). `0` to not notify, `1` to notify all testers that can install this app, `2` to notify all testers
status (optional)
Form Parameter — download status (can only be set with full-access tokens). `1` to not allow users to download the version, `2` to make the version available for download
strategy (optional)
Form Parameter — replace or add build with same build number. `add` to add the build as a new build to even if it has the same build number __default__, `replace` to replace to a build with the same build number
tags (optional)
Form Parameter — restrict download to comma-separated list of tags
teams (optional)
Form Parameter — restrict download to comma-separated list of team IDs
users (optional)
Form Parameter — restrict download to comma-separated list of user IDs
mandatory (optional)
Form Parameter — set version as mandatory `0` for not mandatory __default__, `1` for mandatory
release_type (optional)
Form Parameter — set the release type of the app. `0` for beta __default__ `1` for store `2` for alpha `3` for enterprise
private (optional)
Form Parameter — set to `true` to enable the private download page (default is `true`)
owner_id (optional)
Form Parameter — set to the ID of your organization
commit_sha (optional)
Form Parameter — set to the git commit sha for this build
build_server_url (optional)
Form Parameter — set to the URL of the build job on your build server
repository_url (optional)
Form Parameter — set to your source repository
